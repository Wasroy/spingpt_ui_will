# Dossier Assets

Placez votre logo SpinGPT dans ce dossier.

## Formats supportés

- `logo.png` (recommandé)
- `logo.svg` (pour meilleure qualité)
- `logo.jpg`

Le logo sera automatiquement détecté et affiché dans le header de l'application.

## Taille recommandée

- Hauteur : 48px (sera automatiquement redimensionné si nécessaire)
- Format : PNG avec transparence ou SVG pour meilleure qualité

## Nom du fichier

Le système cherche automatiquement :
1. `assets/logo.png`
2. `assets/logo.svg`
3. `assets/logo.jpg`
4. `logo.png` (racine du projet)
5. `logo.svg` (racine du projet)
